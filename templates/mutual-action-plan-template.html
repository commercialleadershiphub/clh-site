Mutual Action Plan (MAP)

Co-create this plan with your customer. Keep it to one page. Review weekly (15 minutes).

Milestone Target date Seller owner Customer owner Definition of done Status Risk / notes
Security intake submitted Security lead Ticket accepted; review scheduled Data access dependency
Pilot kickoff AE / SE Ops lead Environments ready; success criteria agreed

Weekly notes

Decisions, blockers, next actions, date changes (with reason).

Tip: Link security/legal intake forms and add pilot acceptance criteria inline so decisions feel safe.
